1、升级依赖包：</br>
（1）安装 npm-check</br>
npm install -g npm-check</br>
（2）检查全局 npm 包状态</br>
npm-check -u -g</br>
（3）检查项目 npm 包状态</br>
npm-check -u</br>
（4）通过上下键可以移动光标，使用空格键可以选择需要处理的包，回车直接进行处理
